Polymerizer

Polymerizer is an online application for prediction of surface energy for nanocomposites featuring user-defined polymer structure input and structure-property MQSPR. This web app is in collaboration with and maintained by Rensselaer Polytechnic Institute.

Prediction of Dielectric Constant in Nanodielectrics (currently not available)

This web app simulates the dielectric spectroscopy in nanocomposites with explicit implementation of microstructure dispersion as well as interphase relaxation model. This project is in collaboration with Rensselaer Polytechnic Institute.

Integration of Monte Carlo and Finite Element Analysis in Nanodielectrics (currently not available)

This web app simulates the charge transport in dielectric polymers and their nanocomposites by compling finite element analysis and Monte Carlo. This project is in collaboration with Rensselaer Polytechnic Institute.
